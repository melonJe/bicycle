# 🚲 편한 경로 – 자전거 경로 추천 앱

> **신체 부담을 줄이는 자전거 경로를 추천합니다.**  
> "오늘도 기분 좋게, 무리 없이 라이딩!"

---

## 🔍 소개 (Overview)

**편한 경로**는 단순히 '가장 빠른' 경로가 아니라,  
*노면*, *경사*, *신호 정지*, *차량 교통량* 등을 고려해  
**신체 피로를 최소화하는 '편안한' 자전거 경로**를 추천하는 앱입니다.

---
